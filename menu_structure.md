# 메뉴 구조 요약

## 목적
태일씨앤티 웹사이트의 주요 메뉴 구조를 기준으로 사이트맵을 정리한다.

## 최상위 메뉴
- 회사소개
- 사업실적
- 홍보센터
- ESG경영
- 인재채용
- 관리자

## 상세 메뉴 구조

### 1. 회사소개
- 인사말
- 경영이념
- 회사연혁
- 주거래 시공사
- 업·면허/인증
- 찾아오시는 길

### 2. 사업실적
- 공사수주 현황
- 주택
- 업무시설
- 교육/의료
- 플랜트
- 초고층
- 판매시설
- 기타
- Project Map

### 3. 홍보센터
- News
- 유튜브

### 4. ESG경영
- 하위 메뉴 없음

### 5. 인재채용
- 직무소개
- 인사제도
- 채용가이드
- 복리후생
- 채용FAQ

## 구조 특징
- 전체 메뉴는 1Depth와 2Depth 중심으로 구성된다.
- `사업실적` 메뉴가 가장 많은 하위 항목을 가진다.
- `ESG경영`은 단일 메뉴로 구성된다.
- `홍보센터`는 뉴스와 영상 콘텐츠 중심이다.
- `인재채용`은 채용 정보와 복지 안내 중심으로 구성된다.

```text
└─ 찾아오시는 길

사업실적
├─ 공사수주 현황

├─ 초고층
├─ 판매시설
├─ 기타

홍보센터
├─ News

ESG경영

인재채용

├─ 인사제도

```
