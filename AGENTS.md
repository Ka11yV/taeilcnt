# 태일씨엔티 공식 홈페이지 리뉴얼 프로젝트

## 1. 프로젝트 개요

### 1.1 프로젝트 명
**태일씨엔티 공식 홈페이지 리뉴얼**

### 1.2 프로젝트 목표
태일씨엔티의 공식 홈페이지를 최신 웹 트렌드에 맞게 리뉴얼하고, 사용자 경험을 개선한다.  
기업의 신뢰성과 전문성을 효과적으로 전달할 수 있는 정적 웹사이트를 구축하는 것을 목표로 한다.

### 1.3 추진 배경 및 목적
기존 홈페이지의 노후화된 UI를 개선하고, 외부 방문자와 지원자가 기업 정보를 쉽고 명확하게 확인할 수 있도록 한다.

### 1.4 주요 타겟
- 외부 인사
- 태일씨엔티 지원자
- 기업 정보를 확인하려는 일반 방문자

---

## 2. 디자인 가이드라인

### 2.1 디자인 컨셉
**Trust & Professional**

기업의 신뢰감, 안정성, 전문성을 중심으로 디자인한다.  
건설·기술 기업의 이미지를 고려하여 깔끔하고 정돈된 레이아웃, 명확한 정보 구조, 절제된 컬러 사용을 지향한다.
디자인은 /static/static.png를 참고하여 진행한다.
제작에 사용되는 사진과 영상, 아이콘등은 /static 폴더에 있다.

- 대우건설  
  https://m.daewooenc.com/

- 포스코

단, 외부 라이브러리, 이미지, 폰트 사용 시 저작권에 유의하며 무료 사용이 가능한 리소스를 우선적으로 사용한다.

## 3. 개발 환경

### 3.1 기술 스택
- JavaScript 기반 개발
- Node.js 22 LTS 버전 사용
- 정적 웹사이트 구조로 개발

### 3.2 실행 방식
중간 발표에서는 빌드 전 결과물을 기준으로 확인한다.

```bash
npm install
npm run dev
```

제출 시 `node_modules` 폴더는 제외한다.

### 3.3 라이브러리 사용
외부 라이브러리 사용은 자유롭게 가능하다.  
다만 다음 기준을 준수한다.

- 무료 라이브러리 사용 권장
- 무료 폰트 사용 권장
- 상업적 사용 가능 여부 확인
- 라이선스 및 저작권 문제 주의

## 4. 운영 및 배포 기준

### 4.1 운영 방식
홈페이지 운영은 태일씨엔티에서 관할한다.  
서버 사이드 로직이 없는 정적 호스팅 환경을 고려하여 개발한다.

### 4.2 경로 관리
정적 호스팅 및 발표 환경에서 경로 문제가 발생하지 않도록 절대 경로보다 상대 경로 사용을 권장한다.

예시:

```txt
./assets/images/main.webp
../assets/images/logo.png
```

### 4.3 리소스 관리
모든 이미지, 영상, 폰트 등 필요한 리소스는 프로젝트 폴더 내부에 포함한다.  
외부 CDN 의존도를 최소화하여 제출 환경에서도 정상적으로 동작하도록 구성한다.

---

## 5. 개발 요구사항

### 5.1 성능 최적화
영상과 이미지는 로딩 속도를 고려하여 최적화한다.

권장 포맷:

- 이미지: WebP
- 영상: WebM
- 아이콘: SVG

### 5.2 반응형 웹 구현
모바일, 태블릿, PC 환경에서 화면이 깨지지 않도록 반응형 웹으로 구현한다.

대응 화면:

- Mobile
- Tablet
- Desktop

### 5.3 정적 웹사이트 구조
서버 사이드 로직 없이 정적 웹사이트 형태로 설계한다.  
페이지 이동, 콘텐츠 표시, 메뉴 구성 등은 클라이언트 측에서 처리한다.

### 5.4 메뉴 구조
전체 메뉴 구조는 `menu_structure.md` 파일을 참고하여 구현한다.

---

## 6. 최종 제출물

최종 제출 시 다음 항목을 포함한다.

1. 빌드 결과물
2. 소스 코드 전체
3. 기술서

제출 시 불필요한 의존성 폴더는 제외한다.

제외 권장 항목:

```txt
node_modules/
dist 캐시 파일
.env
임시 작업 파일
```

---

## 7. 개발 시 유의사항

- 기업 홈페이지 특성에 맞게 정보 전달력을 우선한다.
- 과도한 애니메이션보다는 신뢰감 있는 인터랙션을 사용한다.
- 주요 타겟인 외부 인사와 지원자가 원하는 정보를 빠르게 찾을 수 있도록 구성한다.
- 메뉴 구조와 페이지 경로는 명확하게 관리한다.
- 정적 호스팅 환경에서 새로고침, 경로 이동, 이미지 로딩 문제가 발생하지 않도록 주의한다.
- 콘텐츠와 디자인은 태일씨엔티의 기업 이미지와 일관성을 유지한다.
