# 기존 사이트 개요

## 1. 한눈에 보는 현재 사이트 성격

현재 태일씨앤티 사이트는 다음 성격이 강하다.

- **회사 소개형 기업 홈페이지**
- **수주 실적 중심의 신뢰 확보형 구조**
- **홍보 게시판 + 채용 정보 + ESG 정보를 함께 담은 통합형 구조**
- 최신 기업 사이트보다는 **전통적인 게시판/소개 페이지 조합형 웹사이트**에 가깝다.

리뉴얼 관점에서 보면, 현재 사이트는 정보량은 적지 않지만 다음 특징이 함께 보인다.

- 핵심 정보가 여러 메뉴에 분산되어 있음
- 이미지형 콘텐츠와 게시판형 콘텐츠가 혼재되어 있음
- 일부 섹션은 활발하게 운영되지만, 일부는 비어 있거나 중복됨
- 정적 호스팅 중심 리뉴얼 구조와는 맞지 않는 JSP/게시판 의존성이 남아 있음

---

## 2. 현재 상위 메뉴 구조

상위 메뉴는 총 6개다.

1. 회사소개
2. 사업실적
3. 기술혁신
4. 홍보센터
5. 인재채용
6. ESG경영

푸터 및 사이트맵 레이어에는 아래 부가 링크도 노출된다.

- 개인정보처리방침
- 이메일무단수집거부
- 태일 인트라넷
- 태일 NAS
- 태일 원격지원
- 웹메일로 보이는 외부 링크

---

## 3. 홈 화면 구성

홈페이지는 대략 아래 순서로 구성되어 있다.

### 3.1 헤더/내비게이션
- 상단 고정형 내비게이션
- 데스크톱에서는 전체 메뉴 + 사이트맵 버튼
- 모바일에서는 별도 토글 메뉴 사용

### 3.2 메인 비주얼
- 전체 화면형 슬라이드 배너
- `Vegas` 기반 배경 슬라이드 사용
- 확인 시점 기준 주요 문구/비주얼은 최근 수주 프로젝트 이미지 중심
  - 수원연무동 주상복합
  - 다이소 세종 온라인센터
  - 용인 Cluster 1기 OBL
- 메인 슬로건:
  - **“오늘의 안전은 어제로부터, 내일의 안전은 오늘로부터!”**

### 3.3 NEWS 티저
- 최근 홍보 뉴스 3건 노출
- 예시:
  - 삼성물산 공사수행 역량평가 수행우수사 선정, 최우수상 수상
  - 한국청소년육성회 금천지구회 모범청소년 장학금 수여 행사
  - 김경수 대표의 KIBA 서울 6대 회장 취임
- 메인 뉴스 항목은 상세 페이지가 아니라 목록 페이지로 연결되는 구조가 보인다.

### 3.4 수주현황 티저
- 최근 수주 6건 노출
- 우측에는 `Swiper` 기반 프로젝트 이미지 슬라이드가 배치됨
- 최근 대표 프로젝트 예시:
  - 다이소 세종 온라인센터
  - 용인 Cluster 1기 OBL
  - Westin Seoul Parnas 리모델링공사
  - 용인 Cluster 1기 공동구
  - 한국일보용산사옥 신축공사
  - 디지털 엠파이어 평촌 비즈밸리 신축공사

### 3.5 기업 소개 영역
- 유튜브 임베드 영상 1개 + 소개 문구 조합
- “더 나은 내일을 건설하는 기업” 메시지 강조

### 3.6 푸터
- 본사 주소, 전화번호, 팩스
- 정책 페이지 및 업무용 외부 링크 제공

## 4. 메뉴별 콘텐츠 성격

### 4.1 회사소개
- 텍스트형 소개(인사말, 경영이념)
- 연혁/인증/면허 등 신뢰성 자료
- 조직도, 시공사 정보, 위치 안내
- 리뉴얼 시 **기업 신뢰 확보용 핵심 섹션**으로 활용 가치가 높음

### 4.2 사업실적
- 최근 수주 프로젝트 목록
- 건설시공능력 연도별 테이블
- 품질/안전/환경경영 자료
- 실적 페이지가 현 사이트에서 가장 기업 경쟁력을 직접적으로 드러냄

### 4.3 기술혁신
- 비전 설명은 비교적 정리되어 있음
- 하지만 News는 현재 **0건**으로 비어 있음
- 운영 활성도는 낮아 보임

### 4.4 홍보센터
- 현재 사이트에서 가장 활발한 섹션 중 하나
- News는 **전체 180건** 확인
- 사회공헌도 최근 활동이 꾸준히 올라와 있음
- 홍보자료는 eBook/복지 자료/CI 등 4건 수준의 다운로드형 콘텐츠

### 4.5 인재채용
- 인사제도, 복리후생, 채용가이드, FAQ가 존재
- 채용공고는 현재 **0건**
- FAQ는 질문/답변 구조가 잘 정리되어 있어 활용 가치가 높음

### 4.6 ESG경영
- ESG 메뉴는 존재하지만 현재 공개된 정보량은 많지 않음
- ESG/환경경영/윤리경영 탭이 있으나, 공개 내용이 사실상 동일하거나 매우 제한적임
- 리뉴얼 시 독립 구조로 강화할 필요가 큼

## 5. 현재 사이트의 장점

- 연혁, 면허, 인증, 시공사, 실적 등 **신뢰 자산**이 명확함
- 홍보센터와 사회공헌 콘텐츠가 실제 운영되고 있음
- 채용 FAQ, 복리후생, 직무소개는 지원자 관점에서 정보 가치가 있음
- 건설/기술 기업 특유의 **보수적이고 안정적인 인상**은 유지되고 있음

---

## 6. 현재 사이트의 한계

- 이미지 위주 페이지가 많아 정보 재사용성이 낮음
- 메뉴 간 중복/혼선이 있음
  - 사업실적/안전경영/환경경영/ESG경영 경계가 불분명한 부분 존재
- 일부 섹션은 비어 있음
  - 기술혁신 News 0건
  - 채용공고 0건
- 일부 페이지는 게시판/외부 이미지 의존도가 높음
- 전반적으로 반응형/모던 인터페이스보다는 **레거시 기업 사이트 인상**이 강함

---

## 7. 리뉴얼에서 특히 살려야 할 축

기존 사이트 분석 기준으로 리뉴얼의 중심 축은 아래 4가지가 적절하다.

1. **기업 신뢰성**
   - 연혁, 인증, 실적, 시공 파트너
2. **전문성**
   - 사업영역, 프로젝트, 품질/안전 체계
3. **활동성**
   - 홍보센터, 사회공헌, 최신 뉴스
4. **지원자 친화성**
   - 복리후생, FAQ, 직무소개, 채용 흐름

이 4축을 정리해 주면 현재 사이트보다 훨씬 명확한 “Trust & Professional” 방향으로 정리할 수 있다.
